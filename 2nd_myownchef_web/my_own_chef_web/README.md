## 프로젝트 개요

현대인들의 촉박한 시간,
그리고 얇은 지갑을 지키기 위해
하나의 웹 페이지에서

1. 간단한 요리 레시피 정보를 공유
2. 맛있는 간단한 끼니 정보를 공유(편의점)
3. 식비를 절약하기 위해
   식자재 공동 구매를 제공하자는 취지에서
   시작하게 된 프로젝트 입니다.

## 주요 개발 환경

Frontend : HTML/CSS, JavaScript, React
Backend : Java 17(Springboot, Mybatis)
Database : Oracle DB
