# yunjunseo_team_portfolio

## 1st_project

지하철 광고 플랫폼

팀 명 : VertScreen
프로젝트 명 : APM (Advertise Party Maker)
프로젝트에 사용된 주요 기술 : HTML/CSS를 이용한 웹 프로젝트(프론트엔드), 맵API
해당 프로젝트 관련 서류 : 요구사항 정의서, WBS, 화면설계서 및 프로토타입 제작(Figma), 화면시나리오, 시연자료(PPT), 시연영상 제작

프로젝트 인원 : 3 명

## 2nd_project

요리 정보 웹 사이트

팀 명 : Cucook
프로젝트 명 : My Own Chef
프로젝트에 사용된 주요 기술 : React(프론트엔드)를 이용한 웹 프로젝트이며, 백엔드로 Java(springboot프레임워크), 데이터베이스 Oracle DB 사용
해당 프로젝트 관련 서류 : 요구사항 정의서, WBS, 화면설계서 및 프로토타입 제작(Figma), 화면시나리오, 테스트 시나리오, 시연자료(PPT), 시연영상 제작

프로젝트 인원 : 5명

## 3nd_project

2차 My Own Chef를 전면 개조한 간편 요리 레시피 앱

팀 명 : Cucook
프로젝트 명 : My Own Chef
프로젝트에 사용된 주요 기술 : React-native(프론트엔드)를 이용한 하이브리드 앱(안드로이드 한정으로 개발) 프로젝트이며, 백엔드로 Java(springboot프레임워크), 데이터베이스 Oracle DB 사용, Naver Map API, Gemini flash 2.5를 활용한 AI 프롬프트(레시피 자동 추천), Whisper를 활용한 음성인식 + 유튜브 쇼츠 조리법 찾기 등
해당 프로젝트 관련 서류 : 요구사항 정의서, WBS, 화면설계서 및 프로토타입 제작(Figma), 화면시나리오, 테스트 시나리오, 시연자료(PPT), 시연영상 제작

프로젝트 인원 : 5명
